# F-Team 2050 - 웹사이트 배포 가이드

이 웹사이트는 정적 파일(HTML, CSS, JS)로 구성되어 있어 **GitHub Pages**를 이용하면 무료로 간편하게 배포할 수 있습니다.

이미 `docs` 폴더에 배포에 필요한 모든 파일이 정리되어 있으므로, GitHub 설정만 조금 바꿔주면 됩니다.

## 배포 단계 (GitHub Pages)

1. **GitHub 저장소로 이동**: 브라우저에서 이 프로젝트의 GitHub 페이지로 이동합니다.
2. **Settings (설정) 클릭**: 상단 메뉴 중 우측 끝에 있는 `Settings` 탭을 클릭합니다.
3. **Pages (페이지) 메뉴 선택**: 왼쪽 사이드바 메뉴에서 `Pages`를 찾아서 클릭합니다.
4. **Build and deployment (빌드 및 배포) 설정**:
    - **Source**: `Deploy from a branch` 선택
    - **Branch**: `main` 선택
    - **Folder**: `/(root)` 대신 **`/docs`** 를 선택 (중요!)
    - **Save (저장)** 버튼 클릭
5. **배포 완료 대기**:
    - 저장을 누르면 상단에 배포가 시작되었다는 메시지가 뜹니다.
    - 약 1~3분 정도 기다린 후 새로고침을 하면, 상단에 **"Your site is live at..."** 라는 메시지와 함께 사이트 주소가 나타납니다.

## 배포 후 확인

발급된 URL (예: `https://ChoonwooLim.github.io/----/`)로 접속하여 사이트가 정상적으로 보이는지 확인하세요.

## 참고 사항

- **이미지/오디오 경로**: 모든 경로는 상대 경로(`assets/...`)로 설정되어 있어 별도의 수정 없이 작동해야 합니다.
- **업데이트**: 앞으로 코드를 수정하고 `git push`를 하면, 자동으로 변경 사항이 배포된 사이트에 반영됩니다.
